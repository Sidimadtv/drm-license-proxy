🔐 DRM License Proxy

Status: 🟢 Live on Vercel

This proxy server helps bypass CORS restrictions for DRM license requests.

📡 Available Endpoint

URL: /proxy?target=YOUR_TARGET_URL

Example: https://drm-license-proxy.vercel.app/proxy?target=https://license-server.com

🚀 JavaScript Example

const proxyUrl = '/proxy?target=' + encodeURIComponent('https://license-server.com/license');

fetch(proxyUrl, {
    method: 'POST',
    headers: { 'Content-Type': 'application/octet-stream' },
    body: licenseRequestBuffer
});

⚙️ Features

  • ✅ Global Edge hosting on Vercel
  • ✅ CORS enabled (Allow-Origin: *)
  • ✅ Binary data support (Raw Buffers)
